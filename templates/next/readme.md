# Kit + Next.js

[Kit][kit] and [Next.js][next] project template.

## Installation

```sh
npm install --global @compositor/kit
kit init -t next -d my-next-project
```

## Usage

Run the local dev server

```sh
yarn run dev
```

[kit]: https://github.com/c8r/kit
[next]: https://github.com/zeit/next.js
